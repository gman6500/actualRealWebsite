The Computer

In class we had to take apart a computer. It was really fun but at some times a little boring at times but overall fun.

The Parts

RAM

RAM, or Random Access Memory, is used for the storing temporary things that the system needs to now only in the near future.

Processor

The processor is the brain that reads all the information and tells all the other things what to do because of it.

CPU

CPU stands for Computer Processing Unit. This is what processes all the data that the computer throws at it.

Hard Drive

The hard drive is what your computer stores all the information that is required to run progams and such. The actual hard drive is just a cd that can store a large amount of data.

The Computer Cont.

Mother Board

The motherboard is what everything communicates through. If the CPU wants to tell the speakers to do something, it goes through the motherboard and goes to the speaker.

Keyboard

You use this to type and input key commands into the system.

Mouse

The mouse is used to input x and y positioning to allow for better user interface.

Monitor

The monitor allows for a user to see what they are doing inside of the machine so they can actually use the machine.

Power Supply

The power supply is pretty self explanitory. It powers the machine.
