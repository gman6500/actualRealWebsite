Welcome to Code Clicker!
How to play: Start off by clicking the gif of the running code. Congratulations you have just got your first line of code typed! Now keep clicking iuntil you have enough lines of code to purchase the "Macro" on the right. These are buildings, each buliding cost a different amount and writes a different amount of code a second. Buildings will auto-click the gif for you so you can sit down and relax.The more expensive the building, the more it will auto-click. Once you have enough, buy the next building. The "New language". You can click the gif as much as you want so that you can code faster. Now go off and become the best coder the world has ever seen!
