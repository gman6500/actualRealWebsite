Vindinium is a competition where 4 robots compete for gold. Whoever has the most gold after 1200 turns, wins.

Vindinium is played when 4 robots specially designed for it compete against eachother.

Each map is randomly generated. The map has Mines and Taverns.

Each mine provides 1 gold per turn. It takes 20 health to take a mine.

Going to a tavern will heal the hero 50 health. Each robot has 100 health. Each turn takes away 1 health.

Players standing next to eachother wil attack eachother. Attacking a player will cause them to lose 20 health.

If a player is killed by another player, all of their mines are taken by the aggressor.

My bot runs off of an if else if decision heirarchy with function comparation.

It will be: if one of this-->*calling a large amount of function that return true or false* is true, do this, else, check if these things are true so on and so forth.

Using this, my robot can make accurate and situational decisions based on what it is doing.
